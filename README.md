# 📊 SQL Sales Analysis Project

## Overview
This project analyzes sales data using **MySQL**. It covers total sales, profit, category-wise, region-wise, and product-wise analysis using SQL queries.

---

## 📁 Files
| File | Description |
|------|-------------|
| `queries.sql` | Table creation, data insertion, and all analysis queries |
| `README.md` | Project documentation |

---

## 🗃️ Dataset
| Column | Type | Description |
|--------|------|-------------|
| Order_ID | INT | Unique order identifier |
| Date | DATE | Order date |
| Product | VARCHAR | Product name |
| Category | VARCHAR | Product category |
| Region | VARCHAR | Sales region |
| Sales | INT | Sales amount (₹) |
| Profit | INT | Profit amount (₹) |
| Quantity | INT | Units sold |

**Sample Data:** 5 orders | 2 categories | 4 regions | January 2024

---

## 📈 Key Insights
- **Total Sales:** ₹1,01,000
- **Total Profit:** ₹12,200
- **Profit Margin:** 12.1%
- **Top Category:** Electronics (₹78,000)
- **Top Region (Profit):** North (₹5,500)
- **Best Product:** Laptop (₹55,000)

## 🔍 Queries Included
1. Total Sales & Profit
2. Profit Margin %

4. Region-wise Sales & Profit
